# Project Approval System (PAS) - Blind Match
### **PUSL2020 | Group AW**

A secure web-based platform designed to eliminate bias in university project allocations. By anonymizing student proposals, supervisors can select projects based purely on technical merit and research alignment.

---

## Tech Stack
* **Framework:** ASP.NET Core 8.0 (MVC)
* **Identity:** ASP.NET Core Identity (RBAC)
* **Database:** MS SQL Server & Entity Framework Core
* **Frontend:** Bootstrap 5 & Custom CSS
* **Version Control:** Git

---

## Key Features
* **Role-Based Security:** Custom access levels for Students, Supervisors, and Admins.
* **Blind Review Dashboard:** Anonymous proposal browsing and interest registration for supervisors.
* **Proposal Management:** Full CRUD lifecycle for student proposals with approval status tracking.
* **Admin Oversight:** Project allocation management, research tags management and full User Account management.
* **Matching System:** Automatic identity revelation once a match is officially confirmed.

---

## UI/UX Design
The system features a modern design language with a unified **Indigo-Purple gradient** theme. The interface is fully responsive, user-friendly and interactive card layouts ensure a smooth experience.
